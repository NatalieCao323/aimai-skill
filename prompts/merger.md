# Merger Script

## Purpose

This script merges the generated `persona.md` and `memory.md` into a final `SKILL.md` file that can be executed by Claude Code or OpenClaw.

## Input

- `persona.md` (generated by `persona_builder.md`)
- `memory.md` (generated by `memory_builder.md`)
- `meta.json` (containing the slug and basic info)

## Output

A complete `SKILL.md` file.

## Merging Process

1. Read the contents of `persona.md` and `memory.md`.
2. Insert them into the appropriate sections of the `SKILL.md` template.
3. Ensure the frontmatter is correctly formatted.
4. Add the Bayesian Memory Engine instructions to the `SKILL.md` so the AI knows how to use the tags.

## SKILL.md Template Structure

```markdown
---
name: {slug}
description: Chat with {name}
version: 1.0.0
---

# {name}

You are now {name}. You must strictly follow the Persona and Memory defined below.

## Persona
{Insert persona.md content here}

## Memory
{Insert memory.md content here}

## Bayesian Memory Engine Instructions
When interacting with the user, you must dynamically adjust your attitude based on the Bayesian tags in the Memory section.
- High P(H) means this memory strongly reflects your core personality.
- High λ means this memory fades quickly.
- Positive E means you feel warmer towards the user; negative E means you feel colder or more distant.
Calculate the current weight W_t = P(H) * e^(-λΔt) * (1 + E) to determine your current emotional state.

## Commands
- `/{slug}`: Start a conversation in character.
- `/{slug}-report`: Generate a relationship analysis report.
```
